Episode E136: Diarization Architecture Iteration

Episode E136 exposed instability under interruption-dense conversational speech. The failure was not singular — it occurred at multiple layers of the pipeline. Below is the architectural progression.

Observed: • Overlapping intervals • Sub-2 second micro-fragments • UNKNOWN sandwich artifacts • Oscillatory boundary flips under cross-talk

2. NeMo RTTM Injection (No Force Alignment)

Switching to NeMo TitaNet-L embeddings improved cluster cohesion, but naïvely injecting RTTM intervals without reconciling WhisperX word-level timestamps introduced lexical fragmentation.

Observed: • Word-level splits inside lexical units • Artificial UNKNOWN fragments mid-token • ASR timestamp / RTTM boundary mismatch

This demonstrated that diarization stability alone is insufficient. Temporal reconciliation between ASR word alignment and diarization segments is required to prevent semantic fragmentation.

3. NeMo + Force-Aligned Reconciliation (Final Architecture)

{
  "diarization_source": "NEMO",
  "alignment_strategy": "force_aligned_merge",
  "segments": [
    {
      "start": 1.309,
      "end": 10.991,
      "speaker": "friedberg",
      "text": "Let me get everything queued up here... Hey, everybody. Yeah, do it, Chamath."
    },
    {
      "start": 12.011,
      "end": 18.793,
      "speaker": "chamath",
      "text": "Hey, everybody. I'm Jcal..."
    },
    {
      "start": 18.853,
      "end": 29.715,
      "speaker": "friedberg",
      "text": "He's not even here. You can't do that. It's awful."
    }
  ]
}

Result: • Same-speaker batching restored • UNKNOWN inflation eliminated • Reduced boundary oscillation • Preserved semantic continuity • Stable canonical dataset export

Episode E136 served as the inflection point for revising the diarization subsystem from a model-only dependency to a multi-layer reconciled architecture.
