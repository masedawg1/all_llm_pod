Voice Classification Architecture

Speaker Classification Policy Evolution

This page documents the evolution of transcript generation and classification policy. The core constraint: sub-5 second audio is statistically unstable for speaker embedding classification. Version 2 introduced structural gating to enforce a stable embedding regime.

Version Comparison

structural differences
Σ V1 vs V2
V1 — exploratory ~9k files
No duration gating
Classified all segments
Sub-4s included
No propagation system

Risk: classifier operating in instability zone
V2 — disciplined 383 episodes
≥5s classifier policy
Sub-5s marked "lizard"
Cluster-based propagation
Error curve validated empirically

Result: stable embedding domain
