Performance & Instrumentation

The system operates entirely on a local RTX 4070 Super (12GB VRAM) GPU workstation. Latency instrumentation is implemented end-to-end across ASR, diarization, LLM inference, and TTS playback using structured logging and trace-level metrics.

Design Constraints

Operating within a 12GB VRAM ceiling required careful memory management, LoRA hot-swapping, quantized model loading, and selective RAG activation to prevent fragmentation and maintain real-time responsiveness.
