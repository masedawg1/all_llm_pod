Project Background

How This Got Built

The project started as a text style transfer helper — rewriting content in another person's voice using the OpenAI API. Audio forensics pulled it in a different direction.

The All-In Podcast was the obvious corpus: widely followed in technology and venture circles, with a large back-catalog. YouTube provides transcripts, but they aren't usable. A custom transcription pipeline was built from scratch — audio in, structured speaker-labeled text out — and that became the training foundation for each host persona.

V1 ran on Mistral-7B within a couple of months. The personas were believable, but the architecture was fragile: the voice classifier had been trained only on host audio, which meant guest speech could contaminate host training data. The fix was NeMo — more stable clustering, better speaker segmentation than Pyannote. NeMo had been an earlier failed install; once it worked, it became the backbone.

What emerged from that rebuild was Clownfish: a focused labeling service. NeMo generates speaker_id segments; audio grouped by that ID is reviewed by ear and mapped to a name. Once mapped, the assignment propagates through the transcript. Clownfish then expanded beyond labeling into lexical and heuristic discovery — surfacing transcriptional artifacts and speaker misclassifications that would otherwise pass silently into training data.

The pipeline scaled from roughly 185 labeled episodes to more than 530 within a couple of months. Each episode more defensible than the last — not because the models got smarter, but because the process for catching their mistakes did.
