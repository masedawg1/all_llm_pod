ALL-LLM Pod

Multi-Agent LLM System for Real-Time Text & Voice Interaction

ALL-LLM Pod is a full-stack, multi-agent LLM system designed to simulate real-time text and voice interaction between multiple AI personas. The system runs entirely on a local GPU workstation and includes a custom ASR + diarization pipeline, hybrid RAG architecture, Human-in-the-Loop validation tooling, and a SwiftUI client application.

System Architecture

iOS SwiftUI App ↓ (Tailscale) FastAPI Backend (Local GPU Workstation) ├── LLM (Mistral + LoRA Adapters) ├── TTS Engine ├── Hybrid RAG Layer ├── Behavioral Memory Store ├── WhisperX (ASR) ├── NVIDIA NeMo (Diarization) └── Structured Logging & Latency Metrics

ASR & Diarization Pipeline

YouTube URL ↓ yt_dlp ↓ Audio Normalization (16k mono) ↓ WhisperX Transcription ↓ NeMo Diarization ↓ Speaker Merge & Segment Repair ↓ HITL Validation ↓ Speaker-Labeled Dataset Export

Scale
